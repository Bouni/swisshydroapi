Swiss Hydro API - Documentation

Information
  • The data is provided by FEON but they are not responsible for this API by any means.
  • The API data is refreshed every 10 Minutes, which is the maximum allowed polling rate by FEON.
  • Contact me at bouni@owee.de if you need help or have suggestions to improve this API.
  • This API modifys the data slightly in some minor points:
    • It converts Swiss Geo Coordinates to WSG84 because thats most likely more useful for the people unsing it.
    • It translates some text to english because the data set coming from FEON isn't fully translated.
Base URL
https://swisshydroapi.bouni.de/api/v1
